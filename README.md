# Sina Street Food · `sina-street`

Nettside for Sina Street Food (Narvik) — Next.js 15, App Router, Tailwind, Framer Motion.

GitHub-repo: `ben-dik-coder/sina-street`

## Utvikling

```bash
npm install
npm run dev
```

Åpne [http://localhost:3001](http://localhost:3001) (port satt i `package.json`).

## Bygg

```bash
npm run build
npm start
```

## Deploy

[Koble repo til Vercel](https://vercel.com) — bygg: `npm run build`. Lokale bilder: `public/images/` (f.eks. `hero.png`).
